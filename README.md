# Django Portfolio

Animated personal portfolio starter built with Django.

## Quick start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open: http://127.0.0.1:8000/

## Edit your content

Update these files:
- `templates/index.html` for sections and text
- `static/css/style.css` for styles/animation
- `static/js/main.js` for frontend behavior
